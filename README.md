# 📚 LLM Adaptation Techniques – Colab Notebooks

This repository contains hands-on demonstrations of **Large Language Model (LLM) adaptation techniques** implemented using Google Colab notebooks.

---

## 🚀 Contents

### 📓 Notebooks

- 🔗 [Finetuning.ipynb](./Finetuning.ipynb)  
  Demonstrates full model fine-tuning on downstream tasks.

- 🔗 [Transfer_learning.ipynb](./Transfer_learning.ipynb)  
  Covers transfer learning concepts and practical implementation.

- 🔗 [Prompt_engineering.ipynb](./Prompt_engineering.ipynb)  
  Explores prompt design strategies for improving LLM outputs.

---

### 📄 Notes

- 📘 [Notes_prompt_engineering_finetuning.pdf](./Notes_prompt_engineering_finetuning.pdf)  
- 📘 [Notes_transfer_learning.pdf](./Notes_transfer_learning.pdf)

## 📌 How to Use

- Click on any `.ipynb` file above to open it.
- Run the notebooks directly in **Google Colab**.
- Follow step-by-step cells for practical understanding.
